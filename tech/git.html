Git使用相关的问题 #

解决refusing to merge unrelated histories #

今天的使用创建代码的时候，使用github的在线编辑创建了一部分内容。然后在本地编辑又Push到了另一个分支。
在使用git merge master 合并代码的时候遇到了

 blogdist git:(main) git merge master
fatal: refusing to merge unrelated histories

查了一下，没有相关性的两个分支默认是无法合并的，好在git提供了相应的选择，只要使用--allow-unrelated-histories选项即可允许合并,完整命令如

git merge master --allow-unrelated-histories
